Intro to CSS

You can view my Sandbox for this project here.

In 'Intro to Web Design', we learned about cascading style sheets (CSS) and it instantly became one of my favorite parts in designing a website.
At the time I was really into the typeface Courier New, and used it about every chance I could get! I think that typeface teeters between old, typewriter style, yet technical and futuristic. Though accessibility-wise it may not be for everyone.

The Worst Website

You can view my Sandbox for this project here.

For our midterm we created the worst websites to ever exist-- breaking all the rules and ignoring any sense of style, I set out to create the worst e-commerce site there was!
The fun part about this was just going haywire with it and discovering new HTML codes along the way. Such as marquee; in all directions and speed! Finding the most ridiculous images of products also added to the fun.

Website Re-Design
